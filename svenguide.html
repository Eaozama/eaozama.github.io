Настройка раздела Server

Нажмите на кнопку Start New Game

Выберите желаемую карту

Настройка раздела Game

LAN Game - Чекмарка ставится в случае, если вы играете на одной сети, в нашем случае её нужно отключить

HostName - Установите имя "хоста", то-есть сервера, пункт не имеющий почти никакой роли

Max. Players - Установите максимум игроков, которые смогут подключиться

Server Password - Если нужно, вы можете установить пароль на сервер, не обязательное поле

Difficulty Level - Уровень сложности

0 - Дефолт

1 - Легко

2 - Сложно

3 - Реалистично

Server Location - установите локацию сервера

<

Менее важные настройки

Disable explosion gibbing - Вероятно это вам не пригодится

Weapons stay - Оружия остаются

Footsteps - Включить/Выключить Шаги

Autocrosshair - Автоприцел

Intermission Time - Время в секундах перед, значение по умолчанию 6

Disable All Map Time Limits - Отключить Все Ограничения Времени Карт

Disable Low-Player-Count Auto-Balancing - Отключить автобаланс

Allow Voting - Разрешить голосования

Теперь можно начинать, нажмите на кнопку Start

|Подключение к (Listen) серверу

Если вы хост сервера

После загрузки и появления на карте, нажмите клавишу "Ё" для открытия консоли. Пропишите команду status, после вам нужен пункт tcp/ip, копируем. Номер в роде STEAM_0:0:000000000. После чего отправляем его человеку, который должен подключиться к серверу

Если вы подключающийся

После получения строчки с STEAM_0:0:, откройте консоль (Ё) и впишите connect STEAM_0:0:000000000. Это пример с нулями, у вас будет другой номер STEAM_0:0:
