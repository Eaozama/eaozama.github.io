Достижения в CHUCHEL

Автор гайда: .8'13 в Steam

Утречко!

Для получения данного достижения необходимо кликнуть на все предметы с которыми Вы можете взаимодействовать в комнате! (Чтобы понять можно ли с ними взаимодействовать, достаточно навести на них курсор мыши и он видоизменится).

Предметы для взаимодействия:

  • Колонки
  • Будильник
  • Птички
  • Гном
  • Все картины

Бла-Бла-Бла

Для получения данного достижение необходимо трижды нажать на "Желтое существо". Как показано на скриншоте ниже, у "Желтого существа" после нажатия появится кнопка с надписью "BLA".

Поговорите с существом три раза и достижение Ваше!

Любовь к голубым бобам

Для получения данного достижения необходимо взаимодействовать с бобами. Не известно имеет ли смысл какой-то алгоритм, но выполнить я смог это достижения вот таким образом:

  • Съедаем синие бобы
  • Съедаем зеленые и синие бобы
  • Съедаем красные и синие бобы
  • Съедаем оранжевые и синие бобы (Бобы оранжевого цвета сбиваются в соседней левой комнате, где сидит "Большое существо")

Шляпник

Для получения данного достижения необходимо поочередно надевать каждую шляпу и взаимодействовать с каждыми предметами в комнате. Каждая шляпа это разное измерение и в каждом измерение предметы разные с виду, но расположены они одинаково.

Поэтому в стандартной шляпе "Чучела" необходимо взаимодействовать с такими предметами:

  • Радиоприемник (Выполнить все возможные действия)
  • Душ (Выполнить все возможные действия)
  • Клетка с вишней (Выполнить все возможные действия)

В зимней шапке необходимо взаимодействовать с такими предметами:

  • Шар (Выполнить все возможные действия)
  • Сосульки (Выполнить все возможные действия)
  • Снеговик (Выполнить все возможные действия)

И наконец в шапке скомороха необходимо взаимодействовать с такими предметами:

  • Гриб (Выполнить все возможные действия)
  • Красная кнопка (Выполнить все возможные действия)
  • Кактус (Выполнить все возможные действия)

А я тучи разгоню руками

Для получения данного достижения, нужно стрелять по всем облакам.

  • Сначала стреляем по всем облакам и смотрим показанные сцены.
  • Потом стреляем по двум нижним облакам, тем самым соединяем их.
  • Стреляем в большое новое облако и смотрим сцену.
  • Стреляем так же по двум из трех облаков сверху, смотрим сцену.
  • Соединяем большое облако с маленьким верхнем облаком , смотрим сцену.
  • И наконец стреляем в нижнее и верхнее облако и смотрим сцену.

Чтобы облака соединились между собой, необходимо стрелять в свободную область возле облака, например ниже него.

Создатель

Для прохождения данного достижения нужно делать опыты с пробирками. Возможно есть какой-то другой порядок использования пробирок, но я делал так:

1.

Выбираем для обоих персонажей синие капсулы и жидкость из бутылочки слева

Выбираем для обоих одинаковые капли из пробирки "Радость"

2.

Выбираем для обоих синие капсулы и жидкость из бутылочки справа

Выбираем для обоих одинаковые капли из пробирки "Грусть"

3.

Выбираем для обоих синие капсулы и жидкость из бутылочки справа

Выбираем для обоих одинаковые капли из пробирки "Радость"

4.

Выбираем для обоих синие капсулы и жидкость из бутылочки слева

После получения достижения, выбираем для обоих капли из пробирки "Грусть".

Чемпион среди улиток

Данное достижение возможно получить в финальном задании игры "Chuchel". Для этого необходимо выиграть гонку "улиток" у нашего злого зеленого клона.

Для победы нужно нажимать на стрелку, которая будет направлена в правую сторону и на бустеры, которые будут всплывать над клеткой с вишенкой.
